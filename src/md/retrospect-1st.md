# 첫 번째 회고

## 1. 전반적인 소감

최근 **졸업 준비**를 하며 공부를 하지 못해 예전에 공부했던 것들도 잘 기억나지 않을까 봐 걱정이 많았습니다. 하지만 강사님께서 **천천히 차근차근** 알려주셔서 수업을 따라가는 데에는 어려움이 없었습니다.

## 2. 과목별 학습 내용

### 🐙 Git

- 초반에 Git을 다룰 때는 안 한 지 너무 오래되어 '괜찮을까?' 생각했습니다.
- 다행히 강의를 듣자 **예전에 했던 기억들이 떠올라** 무리 없이 따라갈 수 있었습니다.

### 🌐 HTML & CSS

- 이번 주 진도를 나간 HTML과 CSS는 따로 강의를 들으며 예습을 했던 경험이 있습니다.
- 덕분에 진도를 따라가는 데 문제가 없었고, 오히려 **세세한 부분에 더 집중**하며 들을 수 있었습니다.
